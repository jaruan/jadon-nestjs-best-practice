## Introduction

The best practice of nest.js.

## Installation

```bash
npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Development

automatic generate `dto`, `entity`, `controller`, `service`, `module` and the relevant test(`spec`) files.

```bash
nest g <name>

```
